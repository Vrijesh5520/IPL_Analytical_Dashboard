# IPL Analytics Dashboard

* Project Overview

This project presents an interactive Power BI dashboard designed to analyze Indian Premier League (IPL) data. The dashboard helps uncover key performance trends across seasons, venues, teams, and franchises through data visualization and business intelligence techniques.

* Business Problem

Cricket franchises, analysts, and fans often struggle to extract meaningful insights from large volumes of IPL match data.

This dashboard transforms raw IPL data into actionable insights that support:

Team performance evaluation
Venue-based decision making
Toss impact analysis
Season trend analysis
Franchise comparison

* Dashboard Pages :
1. Overview Dashboard

Provides a high-level summary of IPL performance.

KPIs : 
Total Matches
Total Teams
Total Venues
Total Seasons
Win Percentage
Toss Advantage %

Insights :
Overall tournament trends
Match outcome distribution
Toss impact overview

* 2. Season Analysis

Analyzes IPL performance across different seasons.

Key Analysis : 
Matches per season
Seasonal winning trends
Top performing teams
Season-wise comparisons

Business Insight :
Identify dominant teams and changing competitive dynamics across IPL seasons.

* 3. Venue Analysis

Evaluates how different venues influence match outcomes.

Key Analysis : 
Venue-wise matches
Winning patterns by venue
Toss impact by venue
High-performing grounds

Business Insight :
Helps understand venue-specific advantages and strategic conditions.

* 4. Franchise Analysis

Compares team performance across IPL franchises.

Key Analysis :
Matches played
Wins and losses
Win percentage
Toss advantage
Team comparison

Business Insight :
Identifies strongest and most consistent franchises based on historical performance.

* Business Insights Generated :
Toss decisions influence match outcomes but vary across teams and venues.
Certain venues consistently favor specific match strategies.
Franchise performance differs significantly across seasons.
Historical win percentage helps identify consistently successful teams.
Venue conditions impact overall match results and competitiveness.
